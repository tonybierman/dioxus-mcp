dx-playground cockpit

The UI bundle hasn't been built into this server — this is the committed placeholder.

To build and embed the real cockpit UI, then rebuild the server:

crates/dioxus-mcp/scripts/build-ui.sh
cargo build -p dioxus-mcp

For UI iteration without rebuilding the server, run the playground with dx serve and point the server at its output via the DIOXUS_MCP_UI_DIR environment variable.
